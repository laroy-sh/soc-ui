- Grid gaps provide breathing room

- White space reduces cognitive load

- **Body**: 14px, regular
- **Counters**: 42px, bold (attention-grabbing)
- **Labels**: 11-12px, secondary color

### Interaction
- Hover states on navigation
- Active states clearly indicated
- No unnecessary animations
- Fast, responsive feel
